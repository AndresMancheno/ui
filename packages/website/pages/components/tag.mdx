<div />
<div>
  <H2 font="secondary">Tag</H2>
</div>

<Paragraph4>
  Tag is used for describing information in one product, card, etc. Tag
  component admit two props (in addition to inherited from div tag) which are
  <i>kind</i> and <i>textTransform</i> (defaults to <i>uppercase</i>).
</Paragraph4>

<div>
  <H3 font="secondary">Kind</H3>
</div>

<Paragraph4>Possible values primary, secondary, tertiary</Paragraph4>

<Playground>
  <div style={styles}>
    <Tag kind="primary">Primary</Tag>
    <Tag kind="secondary">Secondary</Tag>
    <Tag kind="tertiary">Tertiary</Tag>
  </div>
</Playground>

<div>
  <H3 font="secondary">TextTransform</H3>
</div>

<Paragraph4>
  Possible values uppercase, lowercase, capitalize, none, full-size-kana,
  full-width
</Paragraph4>

<Playground>
  <div style={styles}>
    <Tag kind="primary" textTransform="lowercase">
      Lowercase
    </Tag>
    <Tag kind="secondary" textTransform="capitalize">
      Capitalize
    </Tag>
    <Tag kind="tertiary">Uppercase/Default</Tag>
  </div>
</Playground>

<div>
  <H2 font="secondary">Overrides</H2>
</div>

<Paragraph4>
  You can customize your tag whatever you want, even you can set <i>css </i>
  property to override styles if you need.
</Paragraph4>

<Playground>
  <div style={styles}>
    <Tag css={{ backgroundColor: 'lemonchiffon', color: 'darkblue' }}>
      Override
    </Tag>
  </div>
</Playground>
