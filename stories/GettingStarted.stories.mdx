<div>
  <H3 font="secondary">Install</H3>
</div>

<Paragraph4>Using npm</Paragraph4>

```
npm i @finetwork/ui react
```

<Paragraph4>Or yarn</Paragraph4>

```
yarn add @finetwork/ui react
```

<Paragraph4>
  First declare your theme, then you need to follow the next steps to add styles
  to your app
</Paragraph4>

<div>
  <H3 font="secondary">Client side</H3>
</div>

<Paragraph4>
  Wrap your application with <i>ThemeProvider</i> component from @finetwork/ui
</Paragraph4>

```jsx
export default function App() {
  return (
    <ThemeProvider theme={theme}>
      <YourApp />
    </ThemeProvider>
  )
}
```

<div>
  <H3 font="secondary">Server side</H3>
</div>

<Paragraph4>
  Get css string with method <i>getCssString</i> passing your theme and set css
  string in your style tag
</Paragraph4>

```jsx
export default function App = () => {
  return (
    <>
      <Head>
        <style
          id="fi-ui"
          dangerouslySetInnerHTML={{ __html: getCssString(theme) }}
        />
      </Head>
      <YourApp />
    </>
  )
}
```

</ThemeProvider>
